# XDV Video Maker

XDV Video Maker is a tool designed to facilitate the process of cutting and renaming videos to match the XDV dataset format. With an intuitive interface powered by Streamlit, users can easily select videos from a directory, preview them, and perform cuts according to specified start and end times. Furthermore, the tool allows users to assign up to three labels to the videos, preparing them for inclusion in the XDV dataset.

## Features

- Video selection and previewing through a Streamlit interface.
- Ability to delete videos directly from the interface.
- Video cutting by specifying start and end times.
- Renaming videos to adhere to the XDV dataset naming conventions.
- Assignment of predefined labels to videos.

## Requirements

- Python 3
- Streamlit
- OpenCV

To install the necessary dependencies, run the following command:


    pip install streamlit opencv-python
